## Slide 2: Problem & Solution (Orient)
**문제점 (Problem)**: 
기존의 정적인 식재료 DB는 수작업에 의존하여 정보 갱신이 느리고 모바일 시인성(UX)이 떨어졌음.
**해결책 (Solution)**:
- 3국(한국, 미국, 일본) 공공 API 파이프라인 자동 연동.
- 수작업 제거 및 정보 정확성 100% 보장.

## Slide 3: Bento-Grid UX & Architecture (Learn)
**모듈형 직관성 극대화**
- **Bento-Grid UI**: 정보를 작은 카드 단위로 모듈화하여 화면 스크롤을 최소화, 시인성 극대화.
- **최적의 가독성**: 데이터 전문가들이 장시간 모니터를 보며 피로를 느끼지 않도록 다크 테마 기반 `text-base: 17px` 통일.

## Slide 4: Security & Next Steps (Step)
**네이티브 수준의 최적화**
- **PWA (Progressive Web App)**: 모바일/데스크톱 어디서든 네이티브 앱처럼 동작 및 오프라인 캐싱 지원.
- **Code Obfuscation**: 에셋 난독화 빌드를 적용하여 소스코드 외부 유출 완벽 차단.
